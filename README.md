# Customer Segmentation & Behavioral Analytics

Developed a data-driven customer segmentation framework using Unsupervised Machine Learning to categorize over 2,237 consumers into four distinct behavioral personas. This project enables businesses to transition from "one-size-fits-all" marketing to high-precision, persona-based engagement strategies.

# Methodology & Technical Execution
- Performed extensive EDA and preprocessing, including handling missing values, feature scaling, and Standardization.
- Implemented Principal Component Analysis (PCA) to reduce feature space and improve clustering efficiency.
- Conducted a comparative analysis of K-Means, DBSCAN, and Agglomerative Clustering to determine the most stable group assignments.
- Utilized the Elbow Method and Silhouette Analysis to identify optimal number of clusters.

# Tech Stack
- Language: Python
- Libraries: Scikit-Learn, Pandas, NumPy, Plotly, Seaborn, Matplotlib
- Environment: Google Colab , Jypter Notebook Streamlit

# Results
Validated 4 actionable segments, proving distinct behavioral patterns across 2,237 customers.
